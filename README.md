# HandyLink Mobile Platform 🛠️

HandyLink is a geospatial handyman finder platform designed to bridge the gap between local service providers and households. Whether it's a leaky pipe or a short circuit, HandyLink connects you with the right professionals in your vicinity using real-time location tracking.

This project was developed as part of the **Mobile Application Development Course (Fall 2025)** at **United International University (UIU)**.

---

## 🚀 Key Features

* **Geospatial Discovery:** Find handymen near your current location using integrated map services.
* **Dual-Portal System:** Dedicated interfaces for both Customers and Service Providers (Handymen).
* **Service Categories:** Browse through various categories like Plumbing, Electrical Work, Carpentry, and more.
* **Real-time Interaction:** Seamless communication and booking flow between users.
* **Modern UI/UX:** A responsive and intuitive mobile experience built with React Native.

## ⚙️ Getting Started

### Prerequisites
* Node.js (Latest Stable recommended)
* npm or yarn
* React Native CLI or Expo Go

### Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/xrayian/handylink-mobile-platform.git
    cd handylink-mobile-platform
    ```

2.  **Setup Backend**
    ```bash
    cd express-js-backend
    npm install
    # Configure your .env file
    npm start
    ```

3.  **Setup Frontend**
    ```bash
    cd ../react-native-frontend
    npm install
    npx react-native run-android # or run-ios
    ```
